# Scratch Cart Test App

## Frameworks
- **MVVM Architecture**
- **Hilt** - DI
- **Kotlin Coroutines** - multithreading
- **Jetpack Compose** - UI and Navigation
- **Retrofit** - API calls
- **JUnit4, mockk** - unit tests

To **run the tests**, execute the following command:
```
./gradlew test
```

The project includes static analysis tools **ktlint** and **detekt**. To run them, execute the following command:
```
./gradlew lintAll
```
